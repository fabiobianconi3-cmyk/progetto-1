Formato Tag SOP Lundbeck
Equipment
AAA-YYBBBB-XX
  • AAA = Tipo (PMP, TK, HX...)
  • YY = Anno/Area
  • BBBB = Numero sequenza
  • XX = Suffisso
Linee
AAAA-BB-C-DDDDDD-EEE-FF
  • AAAA = Area
  • BB = Diametro
  • C = Classe specifica
  • DDDDDD = Servizio fluido
  • EEE = Isolamento
  • FF = Sequenza
Strumenti
AAA-BBBBBB-XX
  • AAA = Funzione (PI, LT, TT...)
  • BBBBBB = Numero loop
  • XX = Suffisso
